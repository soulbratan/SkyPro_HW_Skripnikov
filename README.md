## Примеры использования:
1. Для фильтрации списка по ключу 'state' вызовите функцию *'filter_by_currency'* вложив внутрь скобок фильтруемый список словарей.
2. Для сортировки списка по дате вызовите функцию *'sort_by_date()'* вложив внутрь скобок сортируемый список словарей.
3. Для преобразования информации о картах и счетах в модуле *'widget'* создана функция *'mask_account_card'*, которая маскирует введённые данные.
4. В модуле *'widget'* создана функция *'get_date'* которая производит преобразование даты в привычный для человека формат.
5. В модуле *'processing'* создана функция *'filter_by_state'*, фильтрующая входящий список по ключу (по умолчанию: "EXECUTED").
6. В модуле *'processing'* создана функция *'sort_by_date'*, которая сортирует входящий список по дате.
7. В модуле *'generators'* создана функция *'filter_by_currency'*, которая фильтрует входящий список словарей по выбранной валюте.
8. В модуле *'generators'* создана функция *'transaction_descriptions'*. Обрабатывает входящий список словарей и возвращает описание банковской операции.
9. В модуле *'generators'* создана функция *'card_number_generator'*. Генерирует номера банковских карт в заданном диапазоне номеров.

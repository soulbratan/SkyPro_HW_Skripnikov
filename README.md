# Проект SkyPro HW 10.1

## Описание:

*Домашнее задание курса SkyPro, модуля 10.1 (Продвинутый Git).
Проект виджета банковских операций клиента.*

## Установка:

1. **Клонируем репозиторий:**
```
git clone https://github.com/soulbratan/Skypro10_1_SkripnikovOV.git
```
2. **Устанавливаем зависимости:**
```
poetry install
```

## Примеры использования:
1. Для фильтрации списка по ключу 'state' вызовите функцию filter_by_state() вложив внутрь скобок фильтруемый список словарей.
2. Для сортировки списка по дате вызовите функцию sort_by_date() вложив внутрь скобок сортируемый список словарей.

## Примеры использования:
1. Для фильтрации списка по ключу 'state' вызовите функцию *'filter_by_currency'* вложив внутрь скобок фильтруемый список словарей.
2. Для сортировки списка по дате вызовите функцию *'sort_by_date()'* вложив внутрь скобок сортируемый список словарей.
3. Для преобразования информации о картах и счетах в модуле *'widget'* определена функция *'mask_account_card'*, которая маскирует введённые данные.
4. В модуле *'widget'* определена функция *'get_date'* которая производит преобразование даты в привычный для человека формат.
5. В модуле *'processing'* определена функция *'filter_by_state'*, фильтрующая входящий список по ключу (по умолчанию: "EXECUTED").
6. В модуле *'processing'* определена функция *'sort_by_date'*, которая сортирует входящий список по дате.
7. В модуле *'generators'* определена функция *'filter_by_currency'*, которая фильтрует входящий список словарей по выбранной валюте.
8. В модуле *'generators'* определена функция *'transaction_descriptions'*. Обрабатывает входящий список словарей и возвращает описание банковской операции.
9. В модуле *'generators'* определена функция *'card_number_generator'*. Генерирует номера банковских карт в заданном диапазоне номеров.
10. В модуле *'decorators'* определен декоратор *'log'*. Производит логирование функции, её результаты и ошибки. Вывод логов опционально в консоль или файл.
11. В модуле *'utils'* определена функция *'load_transactions_list'*. Преобразует JSON-файл в список словарей.
12. В модуле *'external_api'* определена функция *'exchange_api'*. Запрашивает у Exchange Rates Data API обмен валют по текущему курсу. На вход подаётся с какой на какую валюту и сумма конвертации.
13. В модуле *'utils'* определена функция *'transaction_amount'*. Получает на вход транзакцию и возвращает сумму транзакции в рублях. Если транзакция произведена в другой валюте, то производится конвертация с помощью функции *'exchange_api'*.
